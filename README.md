# Stream Camera — мод для стриминга в OBS

## Что делает

- **Отключает автоматическую паузу** — игра не ставится на паузу, если окно Minecraft потеряло фокус (например, вы переключились на OBS или браузер).
- **Блокирует экран паузы** — меню паузы не появляется при сворачивании окна.
- **Убирает размытие (blur)** при открытии экрана паузы (чистый кадр без эффектов).
- **Циклическая телепортация по точкам** — автоматически перемещает игрока по заранее сохранённым позициям с заданным интервалом. Идеально для красивых облётных камер в спектаторе.
- **Ручное управление** — можно в любой момент переключиться на следующую или предыдущую точку.

## Требования

- Игрок должен быть **оператором (админом)** на сервере или в одиночной игре (через `/op` или включённые читы).
- Рекомендуется режим **spectator** (`/gamemode spectator`) — так телепортация будет плавной и не будет мешать управлению.
- Мод создан для **стриминга через OBS**.

## Управление (клавиши по умолчанию)

| Клавиша | Действие |
|---------|----------|
| **F6**   | Запустить / остановить автоматический цикл телепортации |
| **F7**   | Сохранить текущую позицию и поворот как точку |
| **F8**   | Перезагрузить настройки из конфига (например, интервал) |
| **↑** (стрелка вверх) | Принудительно телепортироваться на следующую точку |
| **↓** (стрелка вниз) | Принудительно телепортироваться на предыдущую точку |

## Настройка

Все файлы хранятся в папке `config/stream_camera/`:

- `settings.toml` — параметры мода (пока только интервал между автоматическими телепортами в секундах).
- `points.toml` — список сохранённых точек (координаты + поворот). Можно редактировать вручную.

## Как использовать

1. Зайдите в мир в режиме **spectator** (или хотя бы с правами оператора).
2. Встаньте в нужную точку, посмотрите в нужную сторону → нажмите **F7** (сохранить точку). Повторите для нескольких точек.
3. Нажмите **F6** — начнётся автоматический проход по точкам с интервалом 2 секунды (можно изменить в конфиге).
4. Если нужно переключиться вручную — используйте **стрелки вверх/вниз**.
5. Когда стрим закончится — нажмите **F6** для остановки.

## Примечания

- При активном цикле телепортации счётчик интервала сбрасывается при ручном переключении.
- Если нет ни одной сохранённой точки, F6 покажет предупреждение.
- Мод работает как на клиенте (одиночная игра), так и на сервере (требуется, чтобы сервер разрешал телепортацию). Для лучшей совместимости используйте на сервере с правами администратора.

> ⚠️ **Важное примечание:** данный мод является **техническим** и изначально создавался для личного сервера. Он может содержать недокументированные баги, несовместимости или некорректное поведение в некоторых окружениях. Используйте на свой страх и риск.
